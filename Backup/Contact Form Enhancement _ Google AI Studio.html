Step 2: Download Library Assets

The intl-tel-input library requires a few files to work locally: the main JavaScript file, a utility script, and a CSS file. It's best to keep these vendor files separate from your own code.

  1. Create a vendor directory inside your js folder: js/vendor/.

  2. Download the JavaScript files:

    • intlTelInput.min.js: Right-click and save this link into your js/vendor/ folder.

    • utils.js: Right-click and save this link into your js/vendor/ folder.

  3. Add the script to your HTML: Before your global.js script tag in your main HTML files, add a line to load the library:

    Generated html
          <!-- Add this before your global.js script tag -->
<script src="js/vendor/intlTelInput.min.js"></script> 
<script src="js/global.js"></script> 
</body>
</html>

### Step 4: Update `global.js`

Now, you will add the initialization code to your `global.js` file. This should go inside the `initializePage` function to ensure it runs after the header (and the rest of the page DOM) has been loaded.

```javascript
// In your global.js file

document.addEventListener('DOMContentLoaded', function() {

    // ... (your existing loadHeaderAndFooter function)
    function loadHeaderAndFooter() {
        // ...
    }

    // --- Function to Initialize All Page Logic ---
    function initializePage() {
        // --- Hamburger Menu Logic ---
        // ... (your existing hamburger logic)

        // --- Dropdown Menu Logic ---
        // ... (your existing dropdown logic)

        // --- Active Nav Link Highlighting ---
        // ... (your existing nav link logic)

        // --- NEW: Phone Number Input Logic ---
        const phoneInputField = document.querySelector("#phone");
        if (phoneInputField) {
            window.intlTelInput(phoneInputField, {
                initialCountry: "auto",
                geoIpLookup: function(callback) {
                    fetch("https://ipapi.co/json")
                      .then(res => res.json())
                      .then(data => callback(data.country_code))
                      .catch(() => callback("us"));
                },
                // Point to the local utils.js file
                utilsScript: "js/vendor/utils.js", 
            });
        }
    }

    // --- Initial call to load header and footer ---
    loadHeaderAndFooter();
});
