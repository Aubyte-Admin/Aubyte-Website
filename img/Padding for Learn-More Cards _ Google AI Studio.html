You've correctly identified that there's an issue with how the text alignment is being applied.

The problem is that text-align: center is a CSS property, but you've placed it directly inside the <p> tag as if it were an HTML attribute, which is incorrect.

To fix this and center the text, you need to apply it as an inline style.

Here is the corrected HTML snippet:

Generated html
      <section id="cta">
    <div class="container text-center">
        <h2>Ready to Partner with Aubyte?</h2>
        <p style="text-align: center;">Discover how the Aubyte Partner Programme can become your strategic advantage.</p>
        <a href="/msp-partner" class="btn btn-primary">Explore the Partner Programme</a>
    </div>
</section>

The key change is:

  • From: <p text-align: center>

  • To: <p style="text-align: center;">

By wrapping the CSS rule within the style="..." attribute, the browser will correctly interpret it and apply the centering to your paragraph.
